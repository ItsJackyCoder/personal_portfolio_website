나에 대한 이야기

대학교와 대학원에서 정보스시템을 전공하며, 백엔드 로직 처리와 웹 UI 디자인에 주된 관심을 두었습니다. 저는 시스템 구조와 사용자 인터페이스 사이의 균형을 찾는 것을 좋아하며, 기능이 원활하게 작동하는 것은 물론 사용자에게도 최적의 경험을 제공하기 위해 노력하고 있습니다.

HTML, CSS

Python(matplotlib,selenium,web crawler,flask)

Java

Database(MySql,MongoDB,SQLite3)

자기소개 어떤 일이든 처음부터 완벽하게 해낼 수는 없다고 생각합니다. 하지만 그 과정에서 남들보다 두세 배 더 노력하는 것을 주저하지 않으며, 이러한 끈기와 노력을 통해 기대 이상의 결과물을 만들어내는 데 강점이 있습니다.

기술 스택
포트폴리오 저는 지금까지 총 5개의 프로젝트를 개발하며 프론트엔드와 백엔드 개발 경험을 모두 쌓았습니다. 이 중에는 자바로 만든 게임 2개, 파이썬과 플라스크 프레임워크 기반의 금융 웹사이트 1개, 그리고 UI 디자인에 집중한 프론트엔드 웹사이트 2개가 포함됩니다. 이처럼 다양한 프로젝트를 통해 시스템 로직 구현과 사용자 인터페이스 설계에 대한 실무 경험을 풍부하게 다질 수 있었습니다.

시작하기 위해 잘할 필요는 없지만 잘하기 위해서는 시작해야만 합니다.
